# AI Study Buddy

This is the repository for the AI Study Buddy project, developed as part of IBE160 Programmering med KI.

## Getting Started

To get started with the project, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/SG-Awesome-like-KI.git
    cd SG-Awesome-like-KI
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
